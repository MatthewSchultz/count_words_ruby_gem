# count_words Ruby Gem
Adds word_count method to string.

## Usage
```ruby
"Some String Content".word_count # => 3
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'count_words'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install count_words
```
